# SpreeTest
Welcome to your new gem! This is built for testing spree decorators

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'spree_test', github: 'softprodigyofficial/spree_test', branch: 'master'
```

And then execute:
    $ bundle install
